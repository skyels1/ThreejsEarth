# ThreejsEarth
### installation & info
to use the site just git clone the repo and then do 
``` npm i ```
in root and then 
``` npx vite ```
to run the site. (site needs vite installed)

this project is just learning threejs (first threejs project)

project uses vite and threejs, that is all.
